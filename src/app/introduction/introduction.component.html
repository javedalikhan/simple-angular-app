Hi this is a sample application built in Angular 6.0.0 (latest).

This small application uses few of the fundamental concepts of angular and is fully responsive.

I have tried to cover the following things

  • Components & Service
  • Property Binding
  • Dependency Injection
  • Routing
  • AJAX requests
  • Responsiveness - Tested in chrome dev tool emulator

The left or top section (in small devices) is the main-content-area. The right or bottom(in small devices) is the notification-area.

The main-content-area shows a component which consumes Notifications Service which is responsible for showing the remarks in the notification-area
